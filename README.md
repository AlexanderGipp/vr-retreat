## Build Dependencies

- [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)

To verify your .NET 6 installation run the following in your terminal:

```bash
dotnet --version
```

## How to Build

Navigate into the `/src` directory of this project and run the following:

```bash
dotnet build
```
